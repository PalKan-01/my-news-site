# Дашборд новостей на Streamlit

Простой дашборд для визуализации данных и отображения новостей.

## Что внутри
- `app.py` — основной код приложения на Streamlit.
- `requirements.txt` — список зависимостей для запуска проекта.

## Как запустить
1. Клонируй репозиторий.
2. Установи зависимости: `pip install -r requirements.txt`.
3. Запусти приложение: `streamlit run app.py`.
